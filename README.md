# 快记 (Quick Task)

> 打开即输入，输完即保存，用完即走

一个极简的任务记录 PWA 应用，专注于「零摩擦」记录体验。

## 功能特性

- ✅ 打开即输入 - 启动后自动聚焦输入框
- ✅ 自动保存 - 输入停顿 2 秒自动保存
- ✅ 离线可用 - PWA 支持离线使用
- ✅ 本地存储 - 数据完全存储在本地
- ✅ 使用统计 - 内置访问统计功能

## 使用方法

1. 直接访问网址
2. 输入你想记录的内容
3. 停顿 2 秒自动保存，或切换到其他应用自动保存
4. 点击右下角统计按钮查看使用数据

## PWA 安装

在移动设备上：
1. 用浏览器打开网址
2. 点击「添加到主屏幕」或「安装应用」
3. 之后可以像原生 App 一样启动

## 技术栈

- 纯 HTML/CSS/JavaScript
- PWA (Service Worker + Web App Manifest)
- localStorage 本地存储
- 无后端依赖

## 部署到 GitHub Pages

```bash
# 1. 创建新仓库
# 2. 将此目录内容推送到 main 分支
# 3. 在仓库设置中启用 GitHub Pages (Source: main branch)
# 4. 访问 https://your-username.github.io/repo-name/
```

## 统计数据导出

用户可以点击统计面板中的「导出数据」按钮，将使用统计数据导出为 JSON 文件。

统计指标包括：
- 总访问次数
- 总记录条数
- 平均首次输入延迟（验证「3秒记录」假设的关键指标）
- 平均每会话记录数
- PWA 启动比例
